# CharacterMakingSystem
キャラクターメイキングを実装するパッケージです。
以下のアセットを利用しています。
- Extenject 9.2.1
- UniRx 7.1.0
- UniTask 2.3.1
- UniVRM 0.103.2
- Starter Assets - Third Person Character Controller 1.1.1
